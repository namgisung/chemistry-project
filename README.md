# 평형상수와 반응지수를 구하고 반응 방향 예측하기
 * 반응지수 = Σ(생성물 계수 × ln[생성물 농도]) - Σ(반응물 계수 × ln[반응물 농도]
 * 평형상수 = = [C]^c * [D]^d / ([A]^a * [B]^b)
   * [A], [B], [C], [D] = 각 반응물과 생성물의 평형 농도
   * a, b, c, d = 각 반응물과 생성물의 계수
